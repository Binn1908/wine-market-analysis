# Analyse du marché du vin américain

Analyse exploratoire du marché du vin aux États-Unis pour accompagner l'entrée sur le marché d'un domaine viticole bourguignon (*Domaine des Croix*).

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://domaine-des-croix-7xcynzbweo4.streamlit.app)

## Contexte

Le Domaine des Croix prépare le lancement de son *Corton Grèves 2016 Pinot Noir* sur le marché américain et souhaite établir un prix compétitif. Ce projet analyse un dataset de 130 000 vins distribués aux États-Unis afin d'identifier les tendances du marché et formuler une recommandation tarifaire.

## Démarche

- Analyse descriptive des notes, prix, régions et cépages
- Visualisation géographique des notes moyennes par pays
- Analyse sémantique des descriptions d'experts (NLP / WordCloud)
- Outil de comparaison de prix filtrable par pays, région, cépage et millésime
- Recommandation tarifaire basée sur les références comparables en Bourgogne

## Stack

Python · Pandas · Matplotlib · Plotly · NLTK · Streamlit
